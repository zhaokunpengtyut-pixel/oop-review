# 🧪 面向对象程序设计 模拟试题（第1套）

> 📌 **题型分布：** 简答题 4题（20分）| 选择题 15题（30分）| 填空题 10空（10分）| 看程序写结果 2题（10分）

## 一、简答题（共4题，每题5分，共20分）

**1.1** 请简述面向对象的四大基本特征，并简要说明每个特征的含义。

**1.2** 什么是构造函数？它有哪些特点？（至少列出4条）

**1.3** 什么是静态成员？静态数据成员和静态成员函数各有什么特点？

**1.4** 什么是友元？友元有哪些优缺点？友元关系有哪些特性？

```cpp

using namespace std;

class Base {
public:
    virtual void show() {
        cout << "Base" << endl;
    }
};

class Derived : public Base {
public:
    void show() override {
        cout << "Derived" << endl;
    }
};

int main() {
    Base* p;
    Base b;
    Derived d;
    p = &b;
    p->show();

    return 0;
}
```

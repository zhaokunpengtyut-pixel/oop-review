## 4. <u>Student(学生)类 — 静态成员/常成员/友元</u>

定义一个学生类，要求如下。
(1)类中的数据成员有学生姓名、学生学号、人数和班级号。至少有一个公有成员函数(Print)。Print负责将相关信息输出。
(2)在类中定义静态数据成员，记录当前学生人数。
(3)在类中定义常数据成员，记录班级号。
(4)添加一个友元函数，可对学生类中的私有数据成员进行访问。（建立两个具体对象，依次输出学生人数和班级号005）
输出示例:
[Student Info] Name: Zhang San | ID: 20260101
[Student Info] Name: Li Si | ID: 20260102
Total Students: 2
Class ID: 005

```cpp
#include <iostream>
#include <string>
#include <iomanip>
using namespace std;
class Student {
private:
    string name;            // 学生姓名
    string id;              // 学生学号
    const int classId;      // 常数据成员 —— 班级号（每个对象创建时确定，不可修改）
    static int count;       // 静态数据成员 —— 当前学生总人数（属于类，不属于单个对象）

public:
    Student(string name, string sid, int cid): name(name), id(sid), classId(cid) {
        count++;            // 创建对象 → 人数 +1
    }

    /**
     * 公有成员函数：输出学生基本信息
     * 格式：[Student Info] Name: xxx | ID: xxx
     */
    void Print() {
        cout << "[Student Info] Name: " << name << " | ID: " << id << endl;
    }

    /**
     * 友元函数声明
     * show 函数虽然不是 Student 的成员，但可以访问其私有成员
     */
    friend void show(Student& s);

    // ---- 公有接口 ----

    /** 静态成员函数：获取当前学生总人数（无需对象，通过类名调用） */
    static int getCount() {
        return count;
    }

    /** 获取班级号（常成员） */
    int getClassId() const {
        return classId;
    }
};

// 初始化静态数据成员（必须在类外定义和初始化）
// 静态成员不属于任何对象，由所有对象共享
int Student::count = 0;

/**
 * 友元函数定义
 * 友元函数不是类的成员函数，但能够访问该类的私有 / 保护成员
 * 此处通过友元 show 直接读取 s 的私有成员 name 和 id
 */
void show(Student& s) {
    cout << "[Student Info] Name: " << s.name << " | ID: " << s.id << endl;
}

int main() {
    // 创建两个学生对象
    // 班级号传入 5，输出时格式化为 005
    Student s1("Zhang San", "20260101", 5);
    Student s2("Li Si",     "20260102", 5);

    // 通过公有成员函数 Print 输出学生信息
    s1.Print();
    s2.Print();

    // 通过静态成员函数获取学生总人数（类名直接调用）
    cout << "Total Students: " << Student::getCount() << endl;

    // 通过友元函数 show 访问私有成员（演示友元的能力）
    // 注意：show 和 Print 的输出格式相同，但 show 不是类成员，
    //       它通过友元特权直接读取了 name 和 id 两个私有成员
    // 输出班级号（常数据成员），setfill / setw 实现 3 位宽度、前导补零

    return 0;
}
```

①动态创建一个CRectangle类的对象,其初始的左下角和右上角坐标分别为(2,5)、(6,8);调用GetPerimeter( )和GetArea( )获得矩形周长和面积，并将周长和面积显示在屏幕上。

③销毁该动态创建的对象。
输出样例：
Perimeter: 14
Area: 12
Perimeter: 12
Area: 9

```cpp
#include <iostream>
#include <cmath>     // 用于 abs() 函数计算绝对值
using namespace std;

// (1) 设计描述平面坐标上的点 CPoint 类
class CPoint {
   private:
    int x,y;
   public:
   CPoint(int x=0,int y=0):x(x),y(y){}
   int GetX()const{
       return x;
   }
   int GetY()const{
       return y;
   }
   void SetX(int x){
       this->x=x;
   }
   void SetY(int y){
       this->y=y;
   }

};

// (2) 设计矩形类 CRectangle
class CRectangle {
    private:
    CPoint zx,ys;
    public:
    CRectangle(const CPoint&zx,const CPoint&ys):zx(zx),ys(ys){
    }
    void SetLPoint(const CPoint&zx){
        this->zx=zx;
    }
    void SetRPoint(const CPoint &ys){
        this->ys=ys;
    }
    int GetPerimeter() const {
        int w = abs(ys.GetX() - zx.GetX());
        int h = abs(zx.GetY() - ys.GetY());
        return 2 * (w + h);
    }

    int GetArea() const {
        int w = abs(ys.GetX() - zx.GetX());
        int h = abs(zx.GetY() - ys.GetY());
        return w * h;
    }


};

// (3) 主函数
int main() {
    CRectangle* rect = new CRectangle(CPoint(2, 5), CPoint(6, 8));
    // 输出初始周长和面积, 以下输出命令供参考
    cout << "Perimeter: " << rect->GetPerimeter() << endl;
    cout << "Area: " << rect->GetArea() << endl;
    rect->SetLPoint(CPoint(4,6));
    rect->SetRPoint(CPoint(7,9));
    cout << "Perimeter: " << rect->GetPerimeter() << endl;
    cout << "Area: " << rect->GetArea() << endl;    
    delete rect;
    return 0;
    }
```

## 6. NUM类 — 连续合数查找

如果某个自然数除了1和它本身之外还有其他因子,则这个自然数就是合数(非素数)。试定义一个类NUM，从3开始向上试探找出n个连续的自然数，且它们都是合数。当找到第一组连续的n个合数后，即停止查找。
具体要求如下:
(1)私有成员:
① int n: 存放满足条件的连续自然数的个数。
② int *p: 根据n的值申请一个动态数组用来存放求出的满足条件的n个自然数。
(2)公有成员函数:
① NUM(int n1): 构造函数，用n1 初始化 n，根据 n的值申请动态数组空间，使p指向该动态数组空间。
② int yes(int x): 判断x是否为合数。是则返回1，否则返回 0。
③ void fun( ): 从3开始向上试探找出n个连续的自然数，且它们都是合数，并依次存入动态数组。
④ void print( ): 输出满足条件的n个合数。
⑤ ~NUM(): 析构函数，释放动态数组所占用的存储空间。
(3)在主函数中定义一个NUM类的对象num，求出10个连续的合数。然后通过对象 num 调用成员函数求出10个连续的合数，并输出计算结果。
注意：这里请注意审题，10个数字是连续的。例如：
1，2，3，4，5 是连续的；
1，4，5，8，18 是不连续的。
输出样例：
The first group of 10 consecutive composite numbers is:
114, 115, 116, 117, 118, 119, 120, 121, 122, 123

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

class NUM {
private:
int n;
int *p;
public:
NUM(int n1) {
        n = n1;
        p = new int[n]; 
}
int yes(int x) {
        if (x <= 1) return 0;
        for (int i = 2; i * i <= x; i++) {
            if (x % i == 0) {
                return 1;
            }
        }
        return 0; // 是素数
    }
 void fun() {
        int count = 0; 
        int i = 3;
        while (count < n) {
            if (yes(i) == 1) {
                p[count] = i;
                count++;         
            } else {
                count = 0;          // 如果遇到非合数（即素数），计数器清零，重新开始找连续的
            }
            i++;
        }
    }
void print() {
        cout << "The first group of " << n << " consecutive composite numbers is:" << endl;
        for (int i = 0; i < n; i++) {
            cout << p[i];
            if (i < n - 1) {
                cout << ", ";
            }
        }
        cout << endl;
}
~NUM() {delete[] p;}
    // 以下输出命令供参考
    // cout << "The first group of " << n << " consecutive composite numbers is:" << endl;
} ;
// (3) 主函数
int main() {
   NUM num(10);
   num.fun();
   num.print();
   return 0;

}
```

## 8. Shape基类派生出Ball/Cylinder/Circular

编写程序，计算球、圆柱和圆锥的表面积和体积。具体要求: 
(1) 定义一个基类，至少含有一个数据成员:半径，并设为保护成员。
(2) 定义基类的三个派生类:球、圆柱、圆锥，都含有求表面积和体积的成员函数和输出函数。
(3) 编写主函数，求球、圆柱、圆锥的表面积和体积。
（半径=2，高为2，PI=3.14）
输出示例：
球的表面积为：50.24
球的体积为：33.49
圆柱的表面积为：50.24
圆柱的体积为：25.12
圆锥的表面积为：30.32
圆锥的体积为：8.37

```cpp
#include <iostream>
#include <cmath>      // sqrt 开平方函数
#include <iomanip>    // fixed / setprecision 格式化输出
using namespace std;

// 圆周率常量（全局常量）
const double PI = 3.14;

/**
 * 抽象基类 —— 形状
 *
 * 包含纯虚函数，因此 Shape 是抽象类，不能实例化
 * 作用：为所有派生类提供统一的接口规范
 */
class Shape {
protected:
    double radius;      // 半径（受保护成员，派生类可直接访问）

public:
    // 构造函数：初始化半径
    Shape(double radius) : radius(radius) {}

    // ---- 纯虚函数（=0 表示纯虚，子类必须重写） ----

    /** 计算表面积 —— 纯虚函数 */
    virtual double getSurfaceArea() const = 0;

    /** 计算体积 —— 纯虚函数 */
    virtual double getVolume() const = 0;

    /** 输出结果 —— 纯虚函数 */
    virtual void output() const = 0;

    // 注意：含有纯虚函数的类是抽象类，不能定义该类的对象
    // 虚析构函数在实际项目中应加上（此处从简）
};

/**
 * 球（Ball）—— 派生自 Shape
 *
 * 表面积公式：S = 4πr²
 * 体积公式：  V = (4/3)πr³
 */
class Ball : public Shape {
public:
    // 构造函数：调用基类 Shape 的构造函数初始化半径
    Ball(double radius) : Shape(radius) {}

    // 重写基类纯虚函数 —— 计算球的表面积
    double getSurfaceArea() const override {
        return 4 * PI * radius * radius;
    }

    // 重写基类纯虚函数 —— 计算球的体积
    // 注意 4.0/3.0 写成浮点数除法，避免整数截断
    double getVolume() const override {
        return (4.0 / 3.0) * PI * radius * radius * radius;
    }

    // 重写基类纯虚函数 —— 输出球的表面积和体积
    void output() const override {
        cout << "球的表面积为：" << fixed << setprecision(2) << getSurfaceArea() << endl;
        cout << "球的体积为："   << fixed << setprecision(2) << getVolume() << endl;
    }
};

/**
 * 圆柱（Cylinder）—— 派生自 Shape
 *
 * 多出一个私有成员：高度 height
 *
 * 表面积公式：S = 2πr² + 2πrh（两个底面积 + 侧面积）
 * 体积公式：  V = πr²h
 */
class Cylinder : public Shape {
private:
    double height;      // 圆柱高度

public:
    // 构造函数：初始化半径（基类）和高度（本类）
    Cylinder(double radius, double height)
        : Shape(radius), height(height) {}

    // 重写纯虚函数 —— 计算圆柱的表面积
    double getSurfaceArea() const override {
        return 2 * PI * radius * radius + 2 * PI * radius * height;
    }

    // 重写纯虚函数 —— 计算圆柱的体积
    double getVolume() const override {
        return PI * radius * radius * height;
    }

    // 重写纯虚函数 —— 输出圆柱的表面积和体积
    void output() const override {
        cout << "圆柱的表面积为：" << fixed << setprecision(2) << getSurfaceArea() << endl;
        cout << "圆柱的体积为："   << fixed << setprecision(2) << getVolume() << endl;
    }
};
/**
 * 圆锥（Circular Cone）—— 派生自 Shape
 *
 * 注：类名 Circular 直译为"圆"，实际代表圆锥（Circular Cone）
 *
 * 表面积公式：S = πr² + πr√(h² + r²) （底面积 + 侧面积）
 * 体积公式：  V = (1/3)πr²h
 */
class Circular : public Shape {
private:
    double height;      // 圆锥高度
public:
    // 构造函数：初始化半径和高度
    Circular(double radius, double height)
        : Shape(radius), height(height) {}
    // 重写纯虚函数 —— 计算圆锥的表面积
    double getSurfaceArea() const override {
        // 底面积 πr² + 侧面积 πr√(h² + r²)
        return PI * radius * radius
             + PI * radius * sqrt(height * height + radius * radius);
    }
    // 重写纯虚函数 —— 计算圆锥的体积
    double getVolume() const override {
        return PI * radius * radius * height / 3;
    }
    // 重写纯虚函数 —— 输出圆锥的表面积和体积
    void output() const override {
        cout << "圆锥的表面积为：" << fixed << setprecision(2) << getSurfaceArea() << endl;
        cout << "圆锥的体积为："   << fixed << setprecision(2) << getVolume() << endl;
    }
};
int main() {
    double radius = 2.0;    // 半径
    double height = 2.0;    // 高度（圆柱 / 圆锥共用）

    // 创建三个派生类对象
    Ball      ball(radius);             // 球
    Cylinder  cylinder(radius, height); // 圆柱
    Circular  circular(radius, height); // 圆锥

    // 通过各自的 output() 输出结果
    // 虽然三个类都有 output()，但各自调用自己的版本
    // 这就是"多态"——同样的接口，不同的行为
    ball.output();
    cylinder.output();
    circular.output();

    return 0;
}
```

## 9. 多层继承：Person→Student/Employee→Faculty/Staff

设计一个 Person类，它有两个派生类Student和Employee，Employee有两个派生类Faculty(教师)和Staff(教工)。

(6) 上述5个类都有一个print()成员函数，用于输出该类对象除薪水外的基本信息。
(7) MyDate类有year、month、day三个数据成员，有一个计算两个MyDate对象间年差的成员函数diffYear(MyDate&)，年差计算只精确到月;三个数据成员的获取函数。
定义上述类，并合理地补充构造函数和其他需要的函数。
分析: 在main()函数中，定义一个Person类对象、一个Student 类对象、一个 Employee类对象、一个Faculty类对象和一个Staff类对象。输出Person类及其派生类对象的基本信息，并输出Faculty类对象和Staff类对象的薪水。

```cpp
#include <iostream>
#include <string>
using namespace std;

class MyDate
{
private:
    int year;
    int month;
    int day;
public:
    MyDate(int y, int m, int d) : year(y), month(m), day(d) {}

    // 计算两个MyDate对象间的年差，精确到月（忽略日）
    double diffYear(const MyDate& other) const
    {
        int years = other.year - year;
        if (other.month < month)

        return years;

    cout << "staff: " << st.getSalary() << endl;
```

## 10. Shape基类派生出Square/Triangle/Circle（虚函数）

声明一个Shape基类，有计算面积Area()成员函数，由此派生出正方形类(Square)、直角三角形类(Triangle)、圆类(Circle)，它们都有Area()成员函数。利用虚函数编程计算正方形、直角三角形和圆的面积之和。（正方形边长为5，直角三角形边长为3、4、5，圆的半径为5）
输出示例：
面积和为：109.5

```cpp
#include <iostream>
using namespace std;

// 1. 定义基类 Shape
class Shape {
public:
    // 定义虚函数 Area，返回 double 类型
    virtual double Area() {
        return 0;
    }
};

// 2. 定义正方形类 Square
class Square : public Shape {
private:
    double side;
public:
    Square(double s) : side(s) {}
    // 重写 Area 函数
    double Area() {
        return side * side;
    }
};

// 3. 定义三角形类 Triangle
class Triangle : public Shape {
private:
    double a, b; // 直角边
public:
    Triangle(double _a, double _b) : a(_a), b(_b) {}
    // 重写 Area 函数
    double Area() {
        return (a * b) / 2.0;
    }
};

// 4. 定义圆形类 Circle
class Circle : public Shape {
private:
    double radius;
public:
    Circle(double r) : radius(r) {}
    // 重写 Area 函数
    double Area() {
        return 3.14 * radius * radius; // 根据题目输出反推，PI取3.14
    }
};

int main() {
    // 题目要求的参数
    Shape *s1 = new Square(5);          // 边长 5
    Shape *s2 = new Triangle(3, 4);     // 直角边 3, 4
    Shape *s3 = new Circle(5);          // 半径 5

    // 利用虚函数计算面积之和
    double sum = s1->Area() + s2->Area() + s3->Area();

    // 输出结果
    cout << "面积和为：" << sum << endl;

    // 释放内存
    delete s1;
    delete s2;
    delete s3;

    return 0;
}
```

## 11. Matrix类 — 运算符重载+和-

编写程序，用成员函数重载运算符“+”和“-”将两个二维数组相加和相减，二维数组的值由构造函数设置。（数组均为两行三列，数组名为X(11,22,33,44,55,66), Y(1,2,3,4,5,6)，求和差的数组名为Z,display输出时设置setw（5））。
输出示例：
please input 2*3 dimension data:
1 2 3 4 5 6
Display object X
   11   22   33
   44   55   66
Display object Y
    1    2    3
    4    5    6
Display object Z=X+Y
   12   24   36
   48   60   72
Display object Z=X-Y
   10   20   30
   40   50   60

```cpp
#include <iostream>
#include <iomanip>  // 提供 setw() 控制输出宽度

using namespace std;

class Matrix {
private:
    int data[2][3];  // 私有成员：2行3列的二维数组

public:
    // 构造函数：用 6 个参数初始化 2×3 矩阵
    // 按"先行后列"的顺序赋值：第一行3个，第二行3个
    Matrix(int a = 0, int b = 0, int c = 0, int d = 0, int e = 0, int f = 0) {
        data[0][0] = a; data[0][1] = b; data[0][2] = c;  // 第一行
        data[1][0] = d; data[1][1] = e; data[1][2] = f;  // 第二行
    }

    // 运算符重载：成员函数形式的 operator+
    // 将当前对象与 another 对应元素相加，返回一个新的 Matrix 对象
    Matrix operator+(const Matrix& other) {
        Matrix temp;  // 临时对象存放结果
        for (int i = 0; i < 2; ++i) {
            for (int j = 0; j < 3; ++j) {
                // this->data 可以省略，直接写 data 也指当前对象的成员
                temp.data[i][j] = this->data[i][j] + other.data[i][j];
            }
        }
        return temp;  // 返回值（不是引用），创建副本传给调用方
    }

    // 运算符重载：成员函数形式的 operator-
    // 将当前对象与 another 对应元素相减
    Matrix operator-(const Matrix& other) {
        Matrix temp;
        for (int i = 0; i < 2; ++i) {
            for (int j = 0; j < 3; ++j) {
                temp.data[i][j] = this->data[i][j] - other.data[i][j];
            }
        }
        return temp;
    }

    // 成员函数：输出矩阵内容，每个元素占 5 个字符宽度
    void display() {
        for (int i = 0; i < 2; ++i) {
            for (int j = 0; j < 3; ++j) {
                cout << setw(5) << data[i][j];  // setw(5)：右对齐，宽度5
            }
            cout << endl;  // 每行输出完换行
        }
    }
};

int main() {
    // 题目要求的 X = {11,22,33,44,55,66}
    Matrix X(11, 22, 33, 44, 55, 66);

    cout << "please input 2*3 dimension data:" << endl;

    // 题目要求的 Y = {1,2,3,4,5,6}
    int a = 1, b = 2, c = 3, d = 4, e = 5, f = 6;
    cout << a << " " << b << " " << c << " " << d << " " << e << " " << f << endl;

    Matrix Y(a, b, c, d, e, f);
    Matrix Z;  // 默认构造，全 0

    cout << "Display object X" << endl;
    X.display();

    cout << "Display object Y" << endl;
    Y.display();

    cout << "Display object Z=X+Y" << endl;
    Z = X + Y;  // 调用 X.operator+(Y)，返回结果赋给 Z
    Z.display();

    cout << "Display object Z=X-Y" << endl;
    Z = X - Y;  // 调用 X.operator-(Y)
    Z.display();

    return 0;
}
```
